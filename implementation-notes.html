Subagent routing implementation notes

  1. Host payload compatibility: Wayland previously replaced every request host payload with its transient executor registration. New requests nest the original payload under that registration. Legacy persisted registrations without a nested payload remain route-less; a request whose route_id claims the Iris route format but lacks a valid nested payload fails closed.
  2. Route reconstruction: The durable route contains provider, model, base URL, and normalized effort only. Provider cache, transport, retry, and context-management settings are reconstructed from the process selection template, then provider-specific context policy and reasoning capability are revalidated. These settings are process-stable today; changing their runtime mutability would require extending the route snapshot contract.
